### 思路
第一反应，计算出链表长度l，倒数第n个节点是正数第l-n+1个节点，这样时间复杂度是O(N),要考虑特殊情况被删掉的是不是头节点，如果是头节点直接返回头节点的next。
官方题解三种解法，计算链表长度、栈、快慢指针。栈就不用了，浪费空间，把快慢指针写了。
快慢指针：一个快指针一个慢指针，快指针先走n步，然后快慢指针一起走，等到快指针走到链尾，慢指针在要删除节点前驱
### 解法一：计算链表长度
```java
class Solution {
    public ListNode removeNthFromEnd(ListNode head, int n) {
        if (head == null)
            return null;
        int l = 0;
        ListNode p = head;
        while (p != null) {
            p = p.next;
            l++;
        }
        //目标节点序号，从0开始
        int target = l - n;
        if (target == 0) {
            return head.next;
        }
        p = head;
        //找到目标节点的前一个节点
        for (int i = 1; i<target; i++) {
            p = p.next;
        }
        p.next = p.next.next;
        return head;
    }
}
```
### 解法二：快慢指针
```java
class Solution {
    public ListNode removeNthFromEnd(ListNode head, int n) {
        if(head == null)
            return null;
        ListNode dummy = new ListNode(0,head);
        ListNode fast = dummy;
        ListNode slow = dummy;
        for(int i = 0;i<=n;i++){
            fast = fast.next;
        }
        while(fast=null){
            slow = slow.next;
            fast = fast.next;
        }
        slow.next = slow.next.next;
        return dummy.next;
    }
}
```
