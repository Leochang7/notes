### 解法一：模拟
#### 思路
用 **四个边界 `firstRow、lastRow、firstColumn、lastColumn` 表示当前还没有被遍历的子矩阵**，每一轮按照 **顺时针顺序依次遍历这一圈的四条边（上 → 右 → 下 → 左）**，把元素加入结果列表；当一圈遍历完成后，将四个边界 **向内收缩一格**（`firstRow++、lastRow--、firstColumn++、lastColumn--`），这样问题规模逐渐缩小，循环继续处理下一层，直到所有元素都被遍历完为止。
#### 代码
```java
class Solution {
    public List<Integer> spiralOrder(int[][] matrix) {
        List<Integer> list = new ArrayList<>();
        int firstRow = 0;
        int lastRow = matrix.length-1;
        int firstColumn = 0;
        int lastColumn = matrix[0].length-1;
        while(firstRow <= lastRow && firstColumn <= lastColumn){
            for(int i = firstColumn;i<=lastColumn;i++){
                list.add(matrix[firstRow][i]);
            }
            for(int i = firstRow+1;i<=lastRow;i++){
                list.add(matrix[i][lastColumn]);
            }
            if(firstRow < lastRow){
                for(int i = lastColumn-1;i>=firstColumn;i--){
                    list.add(matrix[lastRow][i]);
                }
            }
            if(firstColumn < lastColumn){
                for(int i = lastRow-1;i>firstRow;i--){
                    list.add(matrix[i][firstColumn]);
                }
            }

            firstRow++;
            lastRow--;
            firstColumn++;
            lastColumn--;
        }
        return list;
    }
}
```
时间复杂度O(MN) 空间复杂度O(1)
