#二叉树 
#难度：Medium
#独立完成：是
### 解法一
前序遍历数组的第一个元素是根节点，在中序遍历找到这个元素，在这个元素左边的是他的左子树，在他的右边的是他的右子树，在前序遍历他后面的第一个元素是做字数的根节点，左子树元素后的第一个元素的右子树的根节点。可以计算出左子树长度leftSize，然后对左右子树递归操作，左子树的根是preorder里当前根的后一个，右子树的根是preorder里根后leftSize+1个。
```java
class Solution {
    private Map<Integer, Integer> indexMap;
    public TreeNode buildTree(int[] preorder, int[] inorder) {
        indexMap = new HashMap<>();
        for (int i = 0; i < inorder.length; i++) {
            indexMap.put(inorder[i], i);
        }
        return helper(preorder,0,inorder.length - 1,0);
    }

    public TreeNode helper(int[] preorder,int l,int r,int n){
        if (l>r) {
            return null;
        }
        int rootVal = preorder[n];
        TreeNode root = new TreeNode(rootVal);
        int rootIndex = indexMap.get(rootVal);

        int leftSize = rootIndex-l;
        root.left = helper(preorder,l,rootIndex-1,n+1);
        root.right = helper(preorder,rootIndex+1,r,n+leftSize+1);

        return root;
    }
}
```
时间复杂度O(N),空间复杂度O(N)
