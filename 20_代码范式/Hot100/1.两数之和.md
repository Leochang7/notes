### 思路
使用一个HashMap,遍历一遍数组，先计算出target-num[i]，看看这个值是否出现过，如果出现过直接返回下标，如果没出现过把当前元素存入map。要注意，应该先判断再存，如果先存再判断，可能会一个元素用两次。解法两种暴力和哈希，写个最好的哈希就行了。
#### 解法：哈希表
```java
import java.util.HashMap;
class Solution {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer,Integer> map = new HashMap<>();
        for(int i =0;i<nums.length;i++){
            int n = target - nums[i];
            if(map.containsKey(n)){
                return new int[]{i,map.get(n)};
            }
            map.put(nums[i],i);
        }
        return null;
    }
}
```
时间和空间复杂度都是O(N)
