### 思路
第一反应有两种解法，数组和模拟法
先说模拟解法吧，两个链表有长有短，把短的链表对到长链表左端，右边空出来的按0处理，要准备一个变量用于存储进位。处理第一个头节点太烦了，还是建一个新的头节点吧。
数组法还要开辟新数组，没这个省内存，就不写了。
### 方法一：模拟
#### 原代码

```java
class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        ListNode dummy = new ListNode(0);
        ListNode cur = dummy;
        int c = 0;
        while (l1 != null && l2 != null) {
            int val = l1.val + l2.val + c;
            if (val < 10) {
                c = 0;
                cur.next = new ListNode(val);
            } else {
                cur.next = new ListNode(val - 10);
                c = 1;
            }
            l1 = l1.next;
            l2 = l2.next;
            cur = cur.next;
        }
        if (l1 == null) {
            while (l2 != null) {
                int val = l2.val + c;
                if (val < 10) {
                    c = 0;
                    cur.next = new ListNode(val);
                } else {
                    cur.next = new ListNode(val - 10);
                    c = 1;
                }
                l2 = l2.next;
                cur = cur.next;
            }
        }
        if(l2 == null){
            while (l1 != null) {
                int val = l1.val + c;
                if (val < 10) {
                    c = 0;
                    cur.next = new ListNode(val);
                } else {
                    cur.next = new ListNode(val - 10);
                    c = 1;
                }
                l1 = l1.next;
                cur = cur.next;
            }
        }
        if(c>0){
            cur.next = new ListNode(c);
        }
        return dummy.next;

    }
}
```
原来代码太长了，循环可以合并。
#### 优化后代码
```java
class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        ListNode dummy = new ListNode(0);
        ListNode cur = dummy;
        int c = 0;
        while (l1 != null || l2 != null || c!=0) {
            int x = (l1!=null)?l1.val:0;
            int y = (l2!=null)?l2.val:0;
            int sum = x + y + c;
            c = sum / 10;
            cur.next = new ListNode(sum % 10);
            cur = cur.next;
            if (l1 != null) l1 = l1.next;
            if (l2 != null) l2 = l2.next;
        }
        return dummy.next;
    }
}
```
