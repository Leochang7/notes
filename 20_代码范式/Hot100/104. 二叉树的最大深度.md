#二叉树
#难度：Easy
#独立完成：是
### 思路
用一个变量来记录最大深度，使用前序遍历，将上一层的深度也传入下一层，如果下层节点不存在直接返回，如果存在，深度增1，如果大于最大深度，则最大深度+1；
### 解法一：前序遍历
```java
class Solution {
    int maxDepth = 0;
    public int maxDepth(TreeNode root) {
        inOrder(root,1);
        return maxDepth;
    }

    public void preOrder(TreeNode node,int depth){
        if(node == null){
            return;
        }
        
        if(depth>maxDepth){
            maxDepth = depth;
        }

        inOrder(node.left,depth++);
        inOrder(node.right,depth++);
    }
}
```
### 解法二：深度优先
如果已知左右子树的高度，则整棵树的高度为max(l,r)+1，左右子树的高度也可以用同样的方式得到
```java
class Solution {
    public int maxDepth(TreeNode root) {
        if(root == null) return 0;
        int leftHeight = maxDepth(root.left);
        int rightHeight = maxDepth(root.right);
        return Math.max(leftHeight,rightHeight)+1;
    }
}
```
