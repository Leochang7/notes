### 解法一：
给定一个根节点,经过这个根节点的最长路径的长度是左子树的最大深度加右子树的最大深度，然后全局的最长路径不一定结果总树的root，但是总会经过某一个子树的根节点，所以直接后序遍历所有根节点计算经过每个root的最长路径取max。
```java
class Solution {
    int maxD = 0;
    public int diameterOfBinaryTree(TreeNode root) {
        int maxDepth = maxDepth(root);
        return maxD;
    }

    
    public int maxDepth(TreeNode node){
        if(node == null){
            return 0;
        }
        int leftHeight = maxDepth(node.left);
        int rightHeight = maxDepth(node.right);

        if(leftHeight+rightHeight>maxD)
            maxD = leftHeight + rightHeight;

        return Math.max(leftHeight,rightHeight) + 1;
    }
}
```
