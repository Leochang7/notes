### 思路
先是暴力，把所有链表链在一起然后排序。
多路归并，每个链表有一个指针，从头开始，找到所有指针里最小的，然后插入新链表，被选到的指针走一步，重复这个步骤，直到为空。刚开始用的方法不对，用gpt帮忙了。
### 解法一：归并
```java
class Solution {
    public ListNode mergeKLists(ListNode[] lists) {
        ListNode dummy = new ListNode(0);
        ListNode cur = dummy;
        while (true) {
            int idx = -1;
            for (int i = 0; i < lists.length; i++) {
                if (lists[i] == null)
                    continue;
                if (idx == -1 || lists[i].val < lists[idx].val)
                    idx = i;
            }
            if(idx == -1)
                break;
            cur.next = lists[idx];
            lists[idx] = lists[idx].next;
            cur = cur.next;
            cur.next = null;
        }
        return dummy.next;
    }
}
```
