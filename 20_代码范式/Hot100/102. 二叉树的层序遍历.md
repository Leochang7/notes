难度：Medium
是否独立完成：是
### 解法一：前序遍历
把深度信息传进去，进行前序遍历，如果lists的大小等于深度那说明list还没建立，添加进去就行
```java
class Solution {
    List<List<Integer>> lists = new ArrayList<>();
    public List<List<Integer>> levelOrder(TreeNode root) {
        preOrder(root,0);
        return lists;
    }

    public void preOrder(TreeNode node,int depth){
        if(node==null)  return;
        if(lists.size() == depth){
            lists.add(depth,new ArrayList());
        }

        lists.get(depth).add(node.val);
        preOrder(node.left,depth+1);
        preOrder(node.right,depth+1);
    }
}
```
时间复杂度O(n),空间复杂度O(n)
### 解法二：队列

```java
class Solution {
    Queue<TreeNode> queue = new ArrayDeque<>();
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> lists = new ArrayList<>();
        if(root==null) 
            return lists;
        queue.offer(root);
        while(!queue.isEmpty()){
            List<Integer> list = new ArrayList<>();
            int curLevelSize = queue.size();
            for(int i = 0;i<curLevelSize;i++){
                TreeNode node = queue.poll();
                list.add(node.val);
                if(node.left!=null){
                    queue.offer(node.left);
                }
                if(node.right!=null){
                    queue.offer(node.right);
                }
            }
            lists.add(list);
        }
        return lists;
    }

}
```
时间复杂度O(n),空间复杂度O(n)
