#二叉树 
#难度：Medium 
#独立完成：否 

### 解法一：前缀和
在 DFS 过程中，用 **HashMap** 实时记录从根节点到当前节点的累加路径和（前缀和）及其出现次数；每到一个新节点，通过判断 `当前前缀和 - targetSum` 是否在 Map 中存在，来快速定位有多少个祖先节点可以作为起点与当前节点构成目标路径；最后，在递归返回父节点前进行**回溯**，将当前前缀和从 Map 中移除，以确保路径搜索始终保持“自上而下”的合法性。

总结移除的本质：
- **进入节点时**：把自己的“贡献”存入 Map，是为了给自己的**子孙后代**行个方便，让他们能查到。
- **离开节点时**：把自己从 Map 中擦除，是为了**不误导**那些跟自己没血缘关系的“表亲”（其他分支的节点

```java
class Solution {
    Map<Long,Integer> map = new HashMap<>();
    int count = 0;
    public int pathSum(TreeNode root, int targetSum) {
        map.put(0L,1);
        preOrder(root,0,targetSum);
        return count;
    }

    public void preOrder(TreeNode node,long currSum,int targetSum){
        if(node == null){
            return;
        }

        currSum += node.val;

        count += map.getOrDefault(currSum - targetSum, 0);
        map.put(currSum, map.getOrDefault(currSum, 0) + 1);

        preOrder(node.left,currSum,targetSum);
        preOrder(node.right,currSum,targetSum);

        map.put(currSum, map.get(currSum) - 1);
    }
}
```
时间复杂度O(N),空间复杂度O(N)
### 解法二：深度优先搜索
是穷举所有的可能，我们访问每一个节点 node，检测以 node 为起始节点且向下延深的路径有多少种。我们递归遍历每一个节点的所有可能的路径，然后将这些路径数目加起来即为返回结果。
```java
class Solution {

    public int pathSum(TreeNode root, long targetSum) {
        if(root == null)
            return 0;
        int ret = rootSum(root,targetSum);
        ret += pathSum(root.left,targetSum);
        ret += pathSum(root.right,targetSum);
        return ret;
    }

    public int rootSum(TreeNode root,long targetSum){
        int ret = 0;

        if(root == null){
            return 0;
        }

        int val = root.val;
        if(val == targetSum){
            ret++;
        }

        ret += rootSum(root.left,targetSum - val);
        ret += rootSum(root.right,targetSum - val);
        return ret;
    }
}
```
时间复杂度O(N2) 空间复杂度O(N)
