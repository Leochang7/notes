## 1. 目前的想法
### 1.1简单介绍
1.这是为实现《世界种子》预备的第一个项目，饭一口一口吃，路一步一步走
2.游戏类型为文字游戏，在网页上游玩，但是并非一个聊天框
3.后端使用java,前端使用Vue，数据库用SQLlite，前期怎么简单怎么来，RAG数据库使用Chroma
4.在端侧使用几个小模型，是否要上云端模型目前不确定
5.游戏的故事需要构思，目前倾向于武侠小说，我更倾向于故事的发展要符合逻辑，所以应该有个模型负责逻辑审查，不是玩家的所有行为都是合理的，故事可以很离奇，但是要有迹可循
### 1.2游戏的功能
1.游戏可以动态发展，NPC的能力不是一成不变的，NPC的互动也不是一成不变的。举个例子，铁匠铺的铁匠可能会用剑，那我们就可以以某种方式让铁匠教我们练剑，可以让我们学会剑这个技能
2.NPC应该有完整的人生经历，NPC也有一个关系图谱，但是人物是动态的，我们有一个完整的世界观和主线，主线上的人物要在一开始就有，其他的人物需要根据进度生成
3.NPC有人生经历，根据人生经历会有对应的性格，重要的NPC有很多小故事，NPC的可能因为人生中的具体故事对与玩家有相应的交互。
4.与玩家有关的NPC可以更新，不重要的NPC不更新
5.一开始并不知道NPC的性格，是你感觉出来的
