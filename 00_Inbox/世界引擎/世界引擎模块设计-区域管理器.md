# 区域管理器 (Region Manager)

## 1. 模块概述

区域管理器负责管理虚拟世界的三层区域划分（聚光灯区、边缘区、黑盒区），处理区域之间的状态转换，协调不同精度区域之间的交互，确保整个系统的计算资源高效利用。

### 1.1 核心功能

```
世界空间
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│                     区域管理器                               │
├─────────────────────────────────────────────────────────────┤
│  1. 区域划分器 (Region Partitioner)                         │
│     └─ 根据观察者位置和注意力划分区域                        │
│                                                             │
│  2. 状态转换器 (State Transformer)                          │
│     ├─ 具象化（低精度 ▶ 高精度）                            │
│     └─ 抽象化（高精度 ▶ 低精度）                            │
│                                                             │
│  3. 区域协调器 (Region Coordinator)                         │
│     └─ 处理跨区域的事件传播和Agent交互                       │
│                                                             │
│  4. 场变量管理器 (Field Manager)                            │
│     └─ 管理宏观场变量的传播与影响                           │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
三个区域（聚光灯区 / 边缘区 / 黑盒区）
```

### 1.2 区域特性对比

| 特性 | 聚光灯区 (Spotlight) | 边缘区 (Penumbra) | 黑盒区 (Background) |
|------|---------------------|------------------|---------------------|
| 范围 | 玩家视野 ±100m | 视距外 ±1km | 远程区域 |
| Agent数量 | 100-1000 | 1000-10000 | >100000 |
| 精度 | 全精度 | 中精度 | 低精度 |
| Agent状态 | 完整 | 简化 | 统计 |
| 计算方式 | LLM + 行为树 | 行为树 + 马尔可夫链 | 统计方程 |
| 认知时钟 | 1-10s | 10-60s | 分钟/小时 |
| 内存/Agent | >10KB | 1-5KB | <1KB |

### 2.2 划分策略

```python
class RegionPartitioner:
    def __init__(self, config: PartitionConfig):
        self.config = config
        self.spotlight_radius = config.spotlight_radius      # 默认 100m
        self.penumbra_radius = config.penumbra_radius        # 默认 1000m

    def partition(self, observer_pos: Vec3, world_bounds: BoundingBox) -> List[Region]:
        """根据观察者位置划分区域"""
        regions = []

        # 1. 聚光灯区
        spotlight_bounds = self._calculate_circle_bounds(
            observer_pos, self.spotlight_radius
        )
        spotlight_region = Region(
            id=self._generate_id(),
            region_type=RegionType.Spotlight,
            bounds=spotlight_bounds,
            tick_rate=Duration(seconds=1),
            agents=[]
        )
        regions.append(spotlight_region)

        # 2. 边缘区
        penumbra_bounds = self._calculate_annulus_bounds(
            observer_pos,
            inner_radius=self.spotlight_radius,
            outer_radius=self.penumbra_radius
        )
        penumbra_region = Region(
            id=self._generate_id(),
            region_type=RegionType.Penumbra,
            bounds=penumbra_bounds,
            tick_rate=Duration(seconds=10),
            agents=[]
        )
        regions.append(penumbra_region)

        # 3. 黑盒区
        background_bounds = self._calculate_remainder_bounds(
            world_bounds, penumbra_bounds
        )
        background_region = Region(
            id=self._generate_id(),
            region_type=RegionType.Background,
            bounds=background_bounds,
            tick_rate=Duration(minutes=10),
            agents=[]
        )
        regions.append(background_region)

        return regions

    def _calculate_circle_bounds(self, center: Vec3, radius: float) -> BoundingBox:
        """计算圆形区域的边界框"""
        return BoundingBox(
            min=Vec3(center.x - radius, center.y - radius, 0),
            max=Vec3(center.x + radius, center.y + radius, 0)
        )
```

### 2.3 动态调整

区域边界需要根据以下因素动态调整：

```python
class DynamicRegionAdjuster:
    def adjust_regions(self, regions: List[Region], context: AdjustmentContext) -> List[Region]:
        """动态调整区域边界"""
        adjustments = []

        for region in regions:
            # 1. 根据玩家移动预测扩展聚光灯区
            if region.region_type == RegionType.Spotlight:
                predicted_pos = context.player_velocity * context.look_ahead_time
                extended_bounds = self._extend_towards(region.bounds, predicted_pos)
                adjustments.append((region.id, extended_bounds))

            # 2. 根据Agent密度调整区域大小
            agent_density = self._calculate_agent_density(region)
            if agent_density > self.config.density_threshold:
                # 密度过高，缩小区域
                adjustments.append((region.id, self._shrink_region(region)))
            elif agent_density < self.config.density_threshold * 0.5:
                # 密度过低，扩大区域
                adjustments.append((region.id, self._expand_region(region)))

        # 3. 应用调整
        return self._apply_adjustments(regions, adjustments)
```

### 3.2 具象化流程

```python
class StateHydrator:
    def __init__(self, llm_client, memory_store):
        self.llm = llm_client
        self.memory = memory_store

    def hydrate(self, request: HydrationRequest) -> HydrationResult:
        """将低精度Agent转换为高精度Agent"""
        start_time = time.time()

        # 1. 生成基础Agent模板
        agent = self._create_agent_template(request.macro_data)

        # 2. 注入场变量感知
        field_awareness = self._generate_field_awareness(request.field_snapshot)
        agent.perception_fields.update(field_awareness)

        # 3. 生成具象记忆
        memories = self._generate_memories(request.macro_data, request.context)
        for mem in memories:
            self.memory.add(request.agent_id, mem)

        # 4. 调用LLM生成初始状态
        prompt = self._build_hydration_prompt(agent, request.macro_data, request.field_snapshot)
        llm_response = self.llm.generate(prompt)

        # 5. 解析LLM响应
        agent_state = self._parse_agent_state(llm_response)
        agent.state.update(agent_state)

        # 6. 生成初始行为
        agent.current_action = self._generate_initial_action(agent, request.context)

        return HydrationResult(
            agent=agent,
            injected_memories=memories,
            duration=time.time() - start_time
        )

    def _build_hydration_prompt(self, agent: Agent, macro_data: MacroStatistics, fields: Dict) -> str:
        """构建具象化提示词"""
        return f"""
        你正在从休眠中苏醒。请根据以下信息生成你当前的状态：

        角色信息：
        - 姓名：{agent.name}
        - 职业：{agent.occupation}
        - 性格：{agent.personality}

        环境信息：
        - 时间：{macro_data.current_time}
        - 地点：{agent.location}
        - 天气：{macro_data.weather}

        宏观背景：
        - 区域事件：{macro_data.recent_events}
        - 社会氛围：{macro_data.social_mood}
        - 经济状况：{macro_data.economic_status}

        场变量影响：
        {self._format_fields(fields)}

        请生成JSON格式的状态，包含：
        - current_mood: 当前情绪
        - current_goal: 当前目标
        - current_activity: 正在进行的活动
        - immediate_thought: 此刻的想法
        - action_intent: 下一步行动意图
        """

    def _generate_memories(self, macro_data: MacroStatistics, context: Context) -> List[Memory]:
        """根据宏观数据生成具象记忆"""
        memories = []

        # 从宏观事件中提取个体记忆
        for event in macro_data.recent_events:
            if random.random() < event.memory_probability:
                memory = Memory(
                    content=self._personalize_event(event, context),
                    timestamp=event.timestamp,
                    importance=event.importance * random.uniform(0.5, 1.0),
                    tags=["hydration", "macro_derived"]
                )
                memories.append(memory)

        return memories

    def _personalize_event(self, event: Event, context: Context) -> str:
        """将宏观事件个性化为个体记忆"""
        prompt = f"""
        将以下宏观事件转化为个人的第一人称记忆：

        宏观事件：{event.description}
        发生时间：{event.timestamp}
        发生地点：{event.location}

        角色：{context.agent_name}
        职业：{context.agent_occupation}

        请生成一段第一人称的个人记忆，描述这个事件对角色的影响。
        """
        return self.llm.generate(prompt)
```

### 3.3 抽象化 (Dehydration)

当Agent离开高精度区域时，将具象状态压缩为抽象统计数据。

```python
class StateDehydrator:
    def dehydrate(self, agent: Agent) -> MacroStatistics:
        """将高精度Agent转换为统计数据"""
        # 1. 提取关键状态
        key_attributes = {
            "position": agent.position,
            "health": agent.health,
            "inventory_value": self._calculate_inventory_value(agent),
            "mood": agent.mood,
        }

        # 2. 统计历史行为
        behavior_stats = self._analyze_behavior_history(agent)

        # 3. 压缩记忆
        compressed_memories = self._compress_memories(agent.memories)

        # 4. 生成统计数据
        return MacroStatistics(
            position=key_attributes["position"],
            health=key_attributes["health"],
            wealth=key_attributes["inventory_value"],
            mood=key_attributes["mood"],
            behavior_distribution=behavior_stats,
            compressed_memories=compressed_memories,
            last_update=datetime.now()
        )

    def _compress_memories(self, memories: List[Memory]) -> List[CompressedMemory]:
        """压缩记忆为关键标签"""
        # 按重要性排序
        sorted_memories = sorted(memories, key=lambda m: m.importance, reverse=True)

        # 保留最重要的记忆，其他压缩为标签
        important = sorted_memories[:self.config.keep_memories]
        tags = []

        for mem in sorted_memories[self.config.keep_memories:]:
            tags.extend(mem.tags)

        # 统计标签频率
        tag_stats = Counter(tags)

        compressed = [CompressedMemory(
            memory=m for m in important
        )]

        # 添加标签摘要
        compressed.append(CompressedMemory(
            tags=dict(tag_stats.most_common(10))
        ))

        return compressed
```

### 3.4 转换触发条件

```python
class TransitionTrigger:
    def should_hydrate(self, agent: Agent, source_region: Region, target_region: Region) -> bool:
        """判断是否需要具象化"""
        # 1. 距离判断
        distance_to_target = self._calculate_distance(agent, target_region)
        if distance_to_target < self.config.hydration_distance:
            return True

        # 2. 事件触发判断
        if self._has_pending_event(agent, target_region):
            return True

        # 3. 关系链判断
        if self._has_connected_agent(agent, target_region):
            return True

        return False

    def should_dehydrate(self, agent: Agent, source_region: Region, target_region: Region) -> bool:
        """判断是否需要抽象化"""
        # 1. 距离判断
        distance_from_source = self._calculate_distance(agent, source_region)
        if distance_from_source > self.config.dehydration_distance:
            return True

        # 2. 注意力判断
        if not self._is_in_attention(agent):
            return True

        # 3. 持续时间判断
        if self._idle_time(agent) > self.config.idle_threshold:
            return True

        return False
```

```python
class CrossRegionEventBus:
    def __init__(self, message_queue):
        self.queue = message_queue
        self.event_handlers = defaultdict(list)

    def publish(self, event: Event, source_region: RegionId, target_regions: List[RegionId]):
        """发布跨区域事件"""
        # 根据区域类型转换事件格式
        for target_region in target_regions:
            converted_event = self._convert_event(event, source_region, target_region)

            # 发送到目标区域
            message = Message(
                type="cross_region_event",
                source=source_region,
                target=target_region,
                payload=converted_event,
                priority=event.priority
            )
            self.queue.publish(message)

    def _convert_event(self, event: Event, source: RegionType, target: RegionType) -> Event:
        """根据目标区域类型转换事件格式"""
        if target == RegionType.Background:
            # 转换为统计数据
            return self._to_macro_event(event)
        elif target == RegionType.Penumbra:
            # 转换为简化事件
            return self._to_simplified_event(event)
        else:
            # 保持原样
            return event

    def _to_macro_event(self, event: Event) -> Event:
        """转换为宏观统计事件"""
        if event.type == "agent_death":
            return Event(
                type="population_change",
                data={"delta": -1, "reason": "death"},
                location=event.location
            )
        elif event.type == "agent_birth":
            return Event(
                type="population_change",
                data={"delta": 1, "reason": "birth"},
                location=event.location
            )
        # 更多转换规则...
```

### 5.2 场变量传播

```python
class FieldPropagator:
    def propagate(self, field: FieldVariable, dt: float) -> FieldVariable:
        """传播场变量"""
        # 1. 扩散（Diffusion）
        diffused = self._diffusion(field, dt)

        # 2. 平流（Advection） - 如风向影响
        advected = self._advection(diffused, dt)

        # 3. 衰减（Decay）
        decayed = self._decay(advected, dt)

        # 4. 源项（Source/Sink） - 增加或减少场变量
        with_source = self._apply_sources(decayed)

        return with_source

    def _diffusion(self, field: FieldVariable, dt: float) -> FieldVariable:
        """扩散 - 使用有限差分法求解扩散方程"""
        new_grid = field.grid.copy()

        for i in range(1, field.grid.height - 1):
            for j in range(1, field.grid.width - 1):
                # 计算拉普拉斯算子
                laplacian = (
                    field.grid.get(i+1, j) +
                    field.grid.get(i-1, j) +
                    field.grid.get(i, j+1) +
                    field.grid.get(i, j-1) -
                    4 * field.grid.get(i, j)
                )

                # 扩散方程：∂u/∂t = D * ∇²u
                D = field.propagation_speed  # 扩散系数
                new_value = field.grid.get(i, j) + D * laplacian * dt
                new_grid.set(i, j, new_value)

        field.grid = new_grid
        field.last_updated = time.time()
        return field

    def _decay(self, field: FieldVariable, dt: float) -> FieldVariable:
        """衰减"""
        for i in range(field.grid.height):
            for j in range(field.grid.width):
                value = field.grid.get(i, j)
                field.grid.set(i, j, value * (1 - field.decay_rate * dt))

        return field
```

### 5.3 场变量查询

```python
class FieldQuery:
    def get_field_value(self, field: FieldVariable, position: Vec3) -> f64:
        """查询某位置的场变量值"""
        # 双线性插值
        grid_x = int(position.x / field.grid.cell_size)
        grid_y = int(position.y / field.grid.cell_size)

        # 边界检查
        grid_x = max(0, min(grid_x, field.grid.width - 2))
        grid_y = max(0, min(grid_y, field.grid.height - 2))

        # 获取四个角的值
        v00 = field.grid.get(grid_y, grid_x)
        v10 = field.grid.get(grid_y, grid_x + 1)
        v01 = field.grid.get(grid_y + 1, grid_x)
        v11 = field.grid.get(grid_y + 1, grid_x + 1)

        # 计算插值权重
        fx = (position.x - grid_x * field.grid.cell_size) / field.grid.cell_size
        fy = (position.y - grid_y * field.grid.cell_size) / field.grid.cell_size

        # 双线性插值
        return (
            v00 * (1 - fx) * (1 - fy) +
            v10 * fx * (1 - fy) +
            v01 * (1 - fx) * fy +
            v11 * fx * fy
        )
```

```python
            if war_level > 0.5:
                agent.vigilance = min(1.0, agent.vigilance + 0.2)
                agent.income *= 0.8  # 收入降低

        # 3. 将场变量信息注入到Agent的感知中
        agent.perception_fields = field_values

        return agent
```

## 6. 完整区域管理流程

```python
class RegionManager:
    def __init__(self, config: RegionConfig):
        self.config = config
        self.partitioner = RegionPartitioner(config)
        self.hydrator = StateHydrator(config)
        self.dehydrator = StateDehydrator(config)
        self.coordinator = RegionCoordinator(config)
        self.field_manager = FieldManager(config)
        self.regions: Dict[RegionId, Region] = {}

    def update(self, observer_pos: Vec3, dt: float):
        """更新区域管理"""
        # 1. 动态调整区域边界
        self._adjust_regions(observer_pos)

        # 2. 更新场变量
        for region in self.regions.values():
            self.field_manager.propagate(region.fields, dt)

        # 3. 处理状态转换
        self._process_transitions()

        # 4. 协调跨区域交互
        self.coordinator.coordinate()

    def _process_transitions(self):
        """处理所有状态转换"""
        # 收集需要转换的Agent
        transitions = []

        for region in self.regions.values():
            for agent_id in region.agents:
                agent = self.get_agent(agent_id)
                target_region = self._find_target_region(agent.position)

                if target_region.id != region.id:
                    # 需要转换
                    if target_region.region_type == RegionType.Spotlight:
                        transitions.append(("hydrate", agent, region, target_region))
                    elif region.region_type == RegionType.Spotlight:
                        transitions.append(("dehydrate", agent, region, target_region))

        # 执行转换
        for transition_type, agent, from_region, to_region in transitions:
            if transition_type == "hydrate":
                result = self.hydrator.hydrate(HydrationRequest(...))
                to_region.add_agent(result.agent)
            elif transition_type == "dehydrate":
                stats = self.dehydrator.dehydrate(agent)
                to_region.update_agent_stats(agent.id, stats)

            # 移动Agent
            from_region.remove_agent(agent.id)
```

### 7.1 区域池化

```python
class RegionPool:
    def __init__(self):
        self.active_regions: Dict[RegionId, Region] = {}
        self.idle_regions: Dict[RegionId, Region] = {}

    def acquire_region(self, bounds: BoundingBox, region_type: RegionType) -> Region:
        """从池中获取区域"""
        # 尝试复用空闲区域
        for region_id, idle_region in self.idle_regions.items():
            if idle_region.region_type == region_type and idle_region.bounds.contains(bounds):
                del self.idle_regions[region_id]
                self.active_regions[region_id] = idle_region
                return idle_region

        # 创建新区域
        new_region = Region(
            id=self._generate_id(),
            region_type=region_type,
            bounds=bounds,
            agents=[]
        )
        self.active_regions[new_region.id] = new_region
        return new_region

    def release_region(self, region_id: RegionId):
        """释放区域回池中"""
        if region_id in self.active_regions:
            region = self.active_regions[region_id]
            region.clear()  # 清空Agent
            self.idle_regions[region_id] = region
            del self.active_regions[region_id]
```

### 7.2 空间索引

```python
class SpatialIndex:
    def __init__(self, cell_size: float):
        self.cell_size = cell_size
        self.grid: Dict[(int, int), List[AgentId]] = defaultdict(list)

    def insert(self, agent_id: AgentId, position: Vec3):
        """插入Agent到空间索引"""
        cell = self._get_cell(position)
        self.grid[cell].append(agent_id)

    def query_range(self, center: Vec3, radius: float) -> List[AgentId]:
        """查询范围内的所有Agent"""
        results = []
        min_cell = self._get_cell(Vec3(center.x - radius, center.y - radius, 0))
        max_cell = self._get_cell(Vec3(center.x + radius, center.y + radius, 0))

        for i in range(min_cell[0], max_cell[0] + 1):
            for j in range(min_cell[1], max_cell[1] + 1):
                results.extend(self.grid.get((i, j), []))

        return results

    def _get_cell(self, position: Vec3) -> (int, int):
        """计算网格坐标"""
        return (
            int(position.x / self.cell_size),
            int(position.y / self.cell_size)
        )
```
